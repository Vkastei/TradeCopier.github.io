DHL Versandmonitor

Zum Managment von Bestellungen in einem Online Shop, habe ich einen DHL Versandmonitor in Javascript (Electron) programmiert. Dieser greift auf in Datenbanken gespeicherte Bestellungsnummern zu und fragt bestimmte Daten zu diesen mithilfe der DHL-Api ab.

GPT Produktbeschreibung-Ersteller

Um einfach Produktbeschreibungen und Keywords für Amazon etc. zu erstellen, habe ich eine Website erstellt, die eingegebene Keywords nach Verkaufschance sortiert und daraufhin Produktbeschreibungen erstellt. Dafür benutzt das Programm die GPT 3.5 KI der OpenAI-Api mithilfe von Requests in Javascript.

Fokus hier ist die Erstellung von Produktbeschreibungen für Duftkerzen.

Code Fixer

Um Error Fixing in der Programmierung zu erleichtertn, habe ich außerdem ein Tool programmiert, welches automatisch Fehler innerhalb Codes korrigiert. Man muss lediglich das fehlerhafte Stück Code sowie die Fehlermeldung in ein Input Feld kopieren und das Programm entwickelt automatisch den verbesserten Code.

Auch dies habe ich mithilfe von GPT 3.5 und Javascript gemacht mit speziell angefertigten Befehlen zur möglichst effektiven Fehlerkorrigierung. Ich benutze dieses Tool regelmäßig beim Programmieren.
